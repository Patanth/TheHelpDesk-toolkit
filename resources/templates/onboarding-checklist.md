## Before Start Date

### Accounts & Access
- [ ] AD account created (username follows naming convention)
- [ ] Email account created and tested
- [ ] Added to appropriate department security groups
- [ ] VPN access provisioned (if applicable)
- [ ] MFA enrolled
- [ ] Password set — user must change at first login

### Hardware
- [ ] Laptop or desktop assigned and imaged
- [ ] Computer named per naming convention and domain-joined
- [ ] Required software installed (Office, VPN client, etc.)
- [ ] Monitor, keyboard, mouse, docking station ready (if in-office)
- [ ] Headset provided (if required)

### Accounts by Application
- [ ] Microsoft 365 license assigned
- [ ] Teams account active
- [ ] Ticketing system access (if applicable)
- [ ] Any department-specific apps provisioned
- [ ] Shared mailboxes added (if applicable)

---

## Day 1

- [ ] Device handed off to user
- [ ] User logged in successfully
- [ ] Password change completed
- [ ] MFA setup confirmed on user's phone
- [ ] Email confirmed working (send/receive test)
- [ ] VPN connected and tested
- [ ] Printers mapped (if applicable)
- [ ] Network drives mapped (if applicable)
- [ ] User walked through IT basics / where to submit tickets

---

## First Week Follow-Up

- [ ] Check in with user — any issues or access they're still missing?
- [ ] Confirm all required software is installed and licensed
- [ ] Verify no account issues (locks, sync failures)
- [ ] Ticket closed with confirmation from user
