## Day of Departure (or as directed by HR)

### Account Disable — Do These First
- [ ] Disable AD account immediately at scheduled time
- [ ] Change AD account password (prevents any cached session reuse)
- [ ] Remove from all security groups
- [ ] Disable Microsoft 365 license / block sign-in
- [ ] Revoke active MFA sessions and tokens
- [ ] Remove VPN access

### Email
- [ ] Set up Out of Office reply (coordinate message with manager)
- [ ] Forward mailbox to manager or delegate (per policy)
- [ ] Note mailbox retention period per your data policy

### Data & Access
- [ ] Manager notified to review shared files/drives for ownership transfer
- [ ] OneDrive/shared drive access reviewed
- [ ] Any service accounts owned by this user identified and transferred
- [ ] Third-party app access revoked (check what's linked to their SSO)

---

## Hardware Recovery

- [ ] Laptop / desktop retrieved
- [ ] Mobile device wiped and retrieved (if company-owned)
- [ ] Access badge / physical keys returned (coordinate with facilities)
- [ ] Device wiped and re-imaged or held per policy

---

## After 30 Days

- [ ] AD account deleted (or per your retention policy)
- [ ] Mailbox deleted or converted to shared mailbox (per policy)
- [ ] Licenses de-assigned to reclaim for new hires
