# Pre-Escalation Checklist

Before escalating a ticket to Tier 2 or higher, confirm you've completed the following. This protects your ticket quality and prevents unnecessary escalations.

---

## General

- [ ] Ticket has a clear, descriptive title (not just "computer broken")
- [ ] User's name, department, and contact info are recorded
- [ ] Asset tag or computer name is documented
- [ ] Date and time the issue started is noted
- [ ] Issue is reproducible — you've seen it happen, not just described
- [ ] You've asked: *"Has this worked before? What changed recently?"*

---

## Troubleshooting Steps Taken

- [ ] Machine has been rebooted (confirm — don't assume)
- [ ] Relevant scripts have been run (note which ones and the output)
- [ ] Google / KB search performed for the exact error message
- [ ] Relevant troubleshooting guide followed (`troubleshooting/common-issues/`)
- [ ] Issue is isolated to this user/machine (or confirmed to be widespread)

---

## Account Issues (if applicable)

- [ ] AD account status checked (locked, expired, disabled)
- [ ] Password reset attempted if appropriate
- [ ] Group membership verified for the resource in question

---

## Network Issues (if applicable)

- [ ] Network Diagnostics script run and output noted
- [ ] Confirmed if issue is on wired or wireless
- [ ] Confirmed if other users on same segment are affected

---

## Documentation

- [ ] All steps taken are documented in the ticket (not just the solution)
- [ ] Any error messages or codes are copied verbatim into the ticket
- [ ] Screenshots attached if relevant
- [ ] Ticket priority and impact are set correctly

---

## Ready to Escalate

- [ ] You can clearly explain what the issue is, what you've tried, and why it needs Tier 2
- [ ] The ticket is assigned to the correct queue
- [ ] You've notified the user that the ticket is being escalated and set expectations

---

> **Tip:** A good escalation note follows this format:
> *"User [name] is experiencing [issue] since [date]. Steps taken: [list]. Current status: [what happens now]. Suspected cause: [your theory]. Next step needed: [what Tier 2 needs to do]."*
